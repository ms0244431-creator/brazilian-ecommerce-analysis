# brazilian-ecommerce-analysis

## 專案目標 (Project Objective)
本專案利用 Olist 巴西電商平台數據集，深入分析消費者的付款偏好與行為模式。透過資料工程與探索性分析 (EDA)，旨在探討「分期付款期數」對「客單價 (AOV)」的影響，為消費金融策略提供數據洞察。

## 關鍵技術 (Tech Stack)
* **語言**: Python 3.14
* **核心套件**:
  * `pandas`: 執行多表資料整併與 ETL 清洗。
  * `matplotlib` & `seaborn`: 繪製視覺化商業統計圖表。

## 專案執行流程 (Workflow)
1. **資料整合**: 透過 `pd.merge()` 關聯訂單與金流資料表，建立完整的分析維度。
2. **數據清洗**: 針對商業邏輯進行嚴謹檢核，識別並排除異常的「0 期分期」數據（確保計算準確度）。
3. **探索性分析 (EDA)**: 統計支付方式占比，並分析分期與客單價之關聯。
4. **視覺化呈現**: 產出分期期數與平均消費金額 (AOV) 的關聯長條圖。

## 核心商業洞察 (Key Insights)
* **高信用卡依賴**: 信用卡支付占比高達 74%，顯示高價值客戶對信用支付的強勁需求。
* **分期槓桿效應**: 數據呈現顯著的正相關趨勢——期數越長，平均客單價越高。這證實了「分期方案」是提升電商平台高單價商品銷售的重要槓桿。
